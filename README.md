# Gorcli

`gorcli` a cli only API client. It can be used for API exploration and testing.

![screenshot of gorcli](docs/images/cli.png)

## Installation

### Install with Go

```sh
go install github.com/eynopv/gorcli@latest
```

## Usage

```sh
gorcli [command]
```

Read [manual](docs/Manual.md)
